What is API?


An API, or Application Programming Interface, is a set of tools and protocols for building software and applications. It specifies how software components should interact, and allows for communication between different systems. (Set of guidelines or rules that state how applications interact)

APIs can be used to access web-based services such as social media sites, weather services, and online databases. As a product manager, understanding how to use APIs can help you integrate external resources and services into your product, and can also help you understand how to build your own APIs to provide value to other companies and developers.

There are several types of APIs, including web-based APIs, operating system APIs, library APIs, and database APIs. REST (Representational State Transfer) and SOAP (Simple Object Access Protocol) are the two most common types of web APIs.

Components

Mainly there are three main components of an API are Request, Response, and Resource. You would make a requestto a server. The server will return a responsethat will contain data regarding a **resource**. For example, if you want to place a new order for a shirt in an e-commerce store, you would click on the "place order" button. When you click on that button, your browser sends a **request** to the e-commerce server. The e-commerce server will send a **response** to your browser after creating a new order, which is the resourcein this case.

  1. Client: A device that requests data from a server. For example: Web Apps, Mobile Apps.
  2. Server: A computer with software installed which allows performing a service on a network. For example: An application server.
  3. Request: When a client asks a server to send data or process data.
  4. Response: Sent back by the server on each request. A response can contain structured data or an indication that data has been processed.

What is cURL?

cURL is a command-line tool that allows a developer to quickly test an API. So instead of building a UI to see the data returned by an API developers can use cURL, in the command line, to quickly send a request and view the data retuned by the API.

What is Method?

A method refers to the type of request being made by the client to a server. The most common methods are GET, POST, PUT and DELETE. GET method is used to retrieve information from the server, POST method is used to submit information to the server, PUT method is used to update information on the server and DELETE method is used to delete information on the server.
